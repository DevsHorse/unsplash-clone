# Unsplash clone

[Demo link](https://unsplash-clone-ten-beige.vercel.app/ 'Demo link')

## Installation

Install dependencies:

```bash
npm install
```

Rename

```bash
example.env -> .env
```

Set your Unsplash Access Key

```bash
VITE_API_ACCESS_KEY=ACCESS_KEY
```

Run script:

```bash
npm start
```
